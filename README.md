Animal Skin Disease Classifier
Overview
A hybrid deep learning computer vision pipeline designed to classify 6 classes of skin diseases from image data. The system extracts features using a pretrained ResNet50 CNN and utilizes a hyperparameter-tuned Support Vector Machine (SVM) to achieve high-accuracy diagnostics.
